1. Download and unzip the file.
2. Initiate the XAMPP server(localhost).
3. Copy the extracted file and place it in xampp/htdocs.
4. Visit localhost/phpmyadmin in your browser.
5. Create a new database and name it 'somptueuse_cosmetics'
6. Click on the import tab and choose the SQL file located inside the 'database' folder of the source code.
7. Go to localhost/somptueuse_cosmetics/ in your web browser after that.
